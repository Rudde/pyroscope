pyroscope

Python Torrent Tools

PyroScope is a collection of tools for the BitTorrent protocol and especially the rTorrent client.

It offers the following components:

  • CommandLineTools for automation of common tasks, like metafile creation
  • rTorrent extensions like a queue manager and statistics
  • a modern and versatile rTorrent web interface

To get started right away, see the QuickStartGuide. If you want to get a first impression without installing the software, look at the ScreenShotGallery. It's also very easy to WriteYourOwnScripts to automate anything that the standard commands can't do (also see the API documentation). To get in contact and share your experiences with other users of PyroScope, join the pyroscope-users mailing list.
